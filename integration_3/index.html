Step 3 - Analytics and Testing

Once a transaction is closed to LevelUp, the sale should be recorded in the POS software. The mechanism for this will vary by POS.

When you’ve completed the analytics integration, please complete the following QA suite and send the results to pos@thelevelup.com:

LevelUp QA Suite

Prep Phase

  • Download the LevelUp consumer app for iOS, Android and Windows Mobile
  • Check to make sure you have test credit (tap settings/cog - look for Global Credit) or a valid credit card linked
  • Make sure your QR code appears on the main LevelUp screen (tap the arrow in the top left until the LevelUp logo appears)
  • Add custom item to the POS that costs $1.00

Premature Scanning Protocols

Scan Before Sale

  • Scan a QR code at the POS login screen.
    • Is there an error? If so, describe.
  • Can this scan be used to tender a sale?
  • Ring up an item for $1.00
  • Tender LevelUp
  • Process the code that was scanned at the login screen
    • Does the transaction go through?
  • Scan a QR code at the main screen following login.
    • Is there an error? If so, describe.
    • Can this scan be used to tender a sale?
  • Ring up an item for $1.00
  • Tender LevelUp
  • Process the code that was scanned at the login screen
    • Does the transaction go through?

Scan During Sale

  • Scan a QR code at the order-taking screen.
    • Does it add items to the order?
    • Does it interrupt order-taking?
    • Does it input text in a text field?
    • Does this text need to be cleared to continue?
    • Is there an error? If so, describe.
    • Can this scan be used to tender a sale?
  • Ring up an item for $1.00
  • Tender LevelUp
  • Process the code that was scanned at the login screen
    • Does the transaction go through?
  • Scan a QR code on the tender media screen.
    • Does it attempt to “tender” the QR code as a tender “amount”?
    • Is there an error? If so, describe.
    • Can this scan be used to tender a sale?
  • Ring up an item for $1.00
  • Tender LevelUp
  • Process the code that was scanned at the login screen
    • Does the transaction go through?

Error Testing

Bad QR Code

  • Ring up a sale for $1.00
  • Use the following QR code: http://grab.by/hR6O
  • Confirm that the transaction does not go through
  • Confirm that the error message is “Bad QR Code”

No Internet

  • Disconnect internet from server/device
  • Ring up a sale for $1.00
  • Open LevelUp App
  • Tender LevelUp sale
  • Wait for timeout
  • Confirm that the error message is “No internet”

Bad Authentication Token

  • Log in to thelevelup.com
  • Navigate to Control Panel
  • Click “De-Authenticate Devices”
  • Ring up a sale for $1.00
  • Open LevelUp App
  • Tender LevelUp sale
  • Confirm that the transaction does not go through
  • Confirm that the error message is “Bad Authentication Token, Please Login”

Service Not Running (If Applicable)

  • Stop the LevelUp adapter on the server (if applicable)
  • Ring up a sale for $1.00
  • Open LevelUp App
  • Tender LevelUp sale
  • Confirm that the transaction does not go through
  • Confirm that the error message is “Service not running”

  • Tender LevelUp Sale
  • Scan QR Code
  • Close Check
  • Open a new check

Integration stability (if applicable)

System Service

  • Reboot the server.
    • Does the windows service automatically start?
  • Restart the service.
    • Does it restart successfully?
  • If the service crashes, does it restart automatically?
  • Is there verbose logging?
  • Are there human-readable error messages?

System App

  • Is there an easy way to accidentally stop the service? How is it restarted? Does it restart automatically?
  • Is it easy to reboot the service if it is closed accidentally?
  • Is there verbose logging?
  • Where is the log located?
  • Does logging need to be enabled?
  • How is logging enabled?
  • Are there human-readable error messages? List them.
