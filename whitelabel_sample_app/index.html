UI / Sample App Screen Flow:

Note that the SDK is only for sending/retrieving the underlying data. The UI lives in the sample app flow, detailed here. Most new apps may want to start from this point, but existing apps will probably want to pull in individual pieces instead of the entire UI project.

Registration & Login

Registering for LevelUp via email or Facebook, along with logging in to existing accounts, again via email + password combination or Facebook connect. The registration flow also supports custom registration fields.

Pay

Pulling and displaying user’s LevelUp QR code, along with tipping slider. QR code automatically refreshes when the server-side code changes.

Receipt & Push Notification Functionality:

This can include interstitial actions for several campaign types: customer acquisition, Give N Get N, opt-in campaigns (email capture).

Places Browsing

Locations/merchants are generally displayed by proximity. Includes place details with hours and map.

Menu Display

Display any custom web content in-app. This requires separate design and client web hosting.

Transaction History

Retrieving and displaying a user’s transaction history at all or a single merchant.

Refer A Friend Social Functionality

“Give N Get N” option. Give N Get N is a specific campaign type in which a customer can “give” $N at a merchant to a customer who hasn’t been there yet. When that referee redeems the credit, the referrer receives $N as well.

Loyalty Program Conversion

Powerful migration functionality to transfer a user’s existing loyalty rewards / progress into LevelUp. This is useful for moving your old loyalty program to your new one.

Scanning & Claiming Campaign Codes

Integrated and optimized ZXing QR Scanning Library for easy claiming of LevelUp Campaign Codes (printed as QR Code) in printed and/or display material.

Account Management

Allows user to reset password and general LevelUp account holder information.

Credit Card Management

Users can add cards to an account manually or use their phone’s camera to scan cards into their account. Also allows users to set default cards or delete cards.

Forgot Password

Users enter their email address to have a temporary password sent.

Report Issue

Users can submit questions or comments directly to LevelUp support.
