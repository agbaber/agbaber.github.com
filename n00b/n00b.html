Don’t Be A N00b


While the LevelUp integration process is a fairly simple one, there are a few key mistakes that you want to make sure you DON’T make.

DO NOT:

  • Permanently store the merchant's LevelUp login credentials
  • Allow the location_id or merchant_id to be entered manually
  • Print paper receipts

DO:

  • Store the access_token, location_id and merchant_id for use with all orders
  • Get merchant_id and location_id(s) via the users/:id.json endpoint
  • Send SKU data with orders
  • Cache the QR code so the customer can scan at any time
