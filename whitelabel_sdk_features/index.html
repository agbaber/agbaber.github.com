White Label SDK Features

The core SDK detailed below is for use in adding LevelUp to existing apps, or for custom apps not wanting or able to use the sample app flow as a base. The SDK includes client code for iOS and Android to automatically handle talking to the LevelUp servers for the following functions:

  • Registration & Login
  • User Info Retrieval
  • Merchant & Location Information
  • Facebook Account Linking
  • Integration with Server-Side Push
  • Scanning & Claiming Campaign Codes
  • Converting Existing Loyalty Cards
  • Account Management
  • Credit Card Management
  • Transaction History

Registration & Login

Registering for LevelUp via email or Facebook, along with logging in to existing accounts, again via email + password combination or Facebook connect. The registration flow also supports custom registration fields.

User Info Retrieval

Retrieving and displaying data about the authenticated user from LevelUp. This data can include:

  • QR Code Data - including tokenization and rotation
  • User Name
  • User Email (with appropriate permissions)
  • Friend Referral Campaign Data

Merchant & Location Information

Retrieving merchant location listings, savings, loyalty info and available credit by location. Merchants are generally returned by proximity of the nearest merchant location, but can also be returned by category, visited status, and several other parameters.

Facebook Account Linking

Seamless integration into Facebook’s Sign On functionality to enable users to link their LevelUp accounts to Facebook and thus gain access to all of LevelUp’s implicit server-side open-graph sharing functionality.

Integration with Server-Side Push

Linking a LevelUp account with Google Cloud Messaging / Apple Push Notification Services. Push notification receipts.

Scanning & Claiming Campaign Codes

Integrated and optimized ZXing QR Scanning Library for easy claiming of LevelUp Campaign Codes (printed as QR Code) in printed and/or display material. Gives merchants deep analytics (e.g. we know that the ad in the newspaper was scanned this many times and the user came back this many times and transacted at this level.)

Converting Existing Loyalty Cards

Powerful migration functionality to transfer a user’s existing loyalty rewards / progress into LevelUp. This is useful for moving your old loyalty program to your new one.

Account Management

Resetting password and general LevelUp account holder information.

Credit Card Management

Allows card addition, set default, and delete functionality, with client-side encryption.

Transaction History

Retrieving and displaying a user’s transaction history at all or a single merchant.
