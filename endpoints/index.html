/loyalties/:unique_id.json

=======

/loyalties/:unique_id.json

>>>>>>> add whitelabel

This displays the basic information and purchasing habits of a unique customer for a business.

Retrieve Loyalty Information for Unique Customer // Authentication Required: YES // Type: GET

Parameters: unique_id access_token

Sample Request:

curl -G https://api.thelevelup.com/v13/loyalties/12345.json

/oauth/access_token

/oauth/access_token

>>>>>>> add whitelabel

Obtain a new OAuth access_token to be used for all orders.

Retrieve Access Token // Authentication Required: NO // Type: POST

Parameters: client_id username password grant_type

Sample Request:

curl -d "client_id=ac28ba9aakd0913jci9192ad928c83ka7k882ke917la001pl172w3g29oao2lla&grant_type=password&username=user@thelevelup.com&password=password&/oauth/access_token" https://api.thelevelup.com/v13/oauth/access_token

/merchants/:merchant_id/orders.json

Place an order or view list of orders // Authentication Required: YES // Type: GET/POST

Required Parameters: merchant_id access_token order[location_id] order[items] order[spend] order[qr_code_data]
