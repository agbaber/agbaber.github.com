Customer Data APIs

Leverage loyalty data with the LevelUp loyalty API calls. These two endpoints will provide all the information you need to slice & dice the rich data of a successful loyalty program.

/loyalties/:unique_id.json

This displays the basic information and purchasing habits of a unique customer for a business.

Retrieve Loyalty Information for Unique Customer // Authentication Required: YES // Type: GET

Parameters: unique_id access_token

Sample Request:

curl -G https://api.thelevelup.com/v13/loyalties/294247.json?access_token=123456-a12345b12c123456abc12345ab1abc1a123456ab12a1234567a1ab12ab1a12a1

Response:

{
    "loyalty": {
        "average_days_between_visits": 1.1496598639455782,
        "average_spend": "0.23",
        "email": "",
        "first_name": "Andrew",
        "first_order_created_at": "08/06/12",
        "id": 294247,
        "last_name": "B",
        "last_order_created_at": "01/22/13",
        "number_of_visits": 203,
        "total_spend": "34.90"
    }
}
